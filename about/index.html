"Code for Hakodate" のミッションは、ICT（情報通信技術）を使って函館・道南地域の様々な問題を解決することです。

Code for Hakodate は、杉本涼と青田基によって2014年8月30日に設立されました。バスが定時通り来ないことに嫌気が差した当時中学3年生の杉本が、函館の交通機関の時刻表・路線・位置情報のオープンデータ化し、関連するアプリを作りやすい環境を作ろうとしたのが始まりです。その際、株式会社まちづくり五稜郭の代表取締役社長の青田を誘い、Code for Hakodate が設立されました。

2015年2月21日、世界中の都市で行われている"International Open Data Day"の函館開催をきっかけに集まった中村拓也、兵藤允彦と工藤卓也によって設立された "Code for DONAN" も Code for Hakodate と同じミッションで活動していたため2015年4月4日に合併し、新生Code for Hakodate として始動しました。

現在は開発者向けの取り組みだけでなく、スマートフォンのアプリ教室や、Wikipediaの記事を作る「Wikipediaタウン」など、市民向けのワークショップなども積極的に行っております。

スタッフ一覧

杉本涼（すぎもと・りょう）

北海道函館中部高等学校全日制課程に在学中です。どうぞよろしくお願いします。

兵藤允彦（ひょうどう・まさひこ）

公立はこだて未来大学大学院に在学中です。情報推薦の研究をしています。ITの力で函館を日本一の街にするのが目標です。

工藤卓也（くどう・たくや）

公立はこだて未来大学大学院に在学中です。観光とバスに関する研究を行っています。函館でのオープンデータ化を推進していきたいと考えています。

中村拓也（なかむら・たくや）

プログラミングはできませんが、横浜からちょくちょく戻って活動中。ITと地域活性をつなぐ仕事をしています。最近は「オープンデータ自治体サミット」の事務局など。「ITって良く分かんないけど便利・楽しい」といった感じにしていければと思います。

木塚あゆみ（きづか・あゆみ）

公立はこだて未来大学に所属。科学館に展示するインタラクティブなコンテンツを作っています。サッポロオープンラボ(sapporolab.com)や函館Arduino勉強会などでも活動中です。

前田実優（まえだ・みゆう）

公立はこだて未来大学大学院に在学中です。防災情報のオープンデータ化に関する研究をしています。お世話になった函館に恩返しがしたいです。
